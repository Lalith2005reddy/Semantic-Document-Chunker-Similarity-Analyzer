Project Title

Semantic Document Chunker using LangChain

Overview

This project demonstrates semantic text chunking and similarity analysis using embeddings.

Traditional chunking splits text using character length.
This project uses semantic similarity to split documents based on meaning.

Features

✔ Semantic document chunking
✔ Text similarity detection
✔ Interactive UI using Streamlit
✔ Embedding-based NLP processing

Tech Stack

LangChain

HuggingFace Embeddings

Sentence Transformers

Streamlit
